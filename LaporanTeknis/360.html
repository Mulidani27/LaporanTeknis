Panorama 360° Lokasi Wisata

Fitur panorama 360° memberikan pengalaman visual yang interaktif dengan menampilkan lokasi wisata dari berbagai sudut pandang menggunakan teknologi Pannellum.

  • Gambar panorama harus dalam format equirectangular.
  • Tampilan otomatis langsung aktif saat halaman dimuat.
  • Kontrol navigasi seperti zoom dan arah tersedia.
  • Kompas ditampilkan untuk orientasi arah pengunjung.
// Inisialisasi viewer panorama
pannellum.viewer('panorama', {
  type: "equirectangular",
  panorama: "Gambar/360.jpg",
  autoLoad: true,
  compass: true,
  showControls: true,
  preview: "Gambar/360.jpg"
});

Catatan: Pastikan gambar panorama berada di folder Gambar/ dan dalam format 360 derajat (equirectangular).

Contoh Elemen HTML Viewer

<div id="panorama" class="w-full h-96 rounded-md"></div>
