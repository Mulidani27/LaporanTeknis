Mengambil Data Cuaca

Setelah Anda memiliki API Key dari OpenWeatherMap, Anda dapat mengambil data cuaca menggunakan JavaScript dengan fetch(). Data akan dikembalikan dalam format JSON dan bisa ditampilkan ke UI.

Langkah-langkah mengambil data cuaca:

  • Tentukan koordinat lintang dan bujur lokasi wisata.
  • Buat URL API dengan menyisipkan API Key dan koordinat.
  • Gunakan fungsi fetch() untuk memanggil API.
  • Ambil dan tampilkan data dari respon JSON.
// Ganti dengan API Key OpenWeatherMap Anda
const apiKey = 'your-api-key';
const latitude = -3.3178;
const longitude = 115.3213;

const url = `http://api.openweathermap.org/data/2.5/weather?lat=${latitude}&lon=${longitude}&appid=${apiKey}&units=metric&lang=id`;

fetch(url)
  .then(response => response.json())
  .then(data => {
    const weather = data.weather[0].description;
    const temperature = data.main.temp;
    const humidity = data.main.humidity;
    const windSpeed = data.wind.speed;

    console.log(`Cuaca: ${weather}`);
    console.log(`Suhu: ${temperature}Â°C`);
    console.log(`Kelembapan: ${humidity}%`);
    console.log(`Kecepatan Angin: ${windSpeed} m/s`);
  })
  .catch(error => console.error('Error:', error));

Catatan: Gunakan protokol https jika Anda menjalankan aplikasi dari server HTTPS agar tidak terjadi blokir konten campuran oleh browser.
