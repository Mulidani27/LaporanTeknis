Penambahan Kontrol Peta

Mapbox menyediakan beberapa kontrol tambahan yang dapat ditambahkan ke peta untuk meningkatkan pengalaman pengguna, seperti:

  • FullscreenControl: Memungkinkan pengguna membuka peta dalam tampilan layar penuh.
  • GeolocateControl: Menampilkan posisi pengguna saat ini pada peta.
// Fullscreen
const fullscreenControl = new mapboxgl.FullscreenControl();
map.addControl(fullscreenControl, 'top-left');

// Geolocate
map.addControl(new mapboxgl.GeolocateControl({
  positionOptions: { enableHighAccuracy: true },
  trackUserLocation: true
}));

Catatan: Anda dapat menempatkan kontrol di berbagai posisi seperti top-left, top-right, bottom-left, atau bottom-right.
