Penambahan Marker Lokasi Wisata

Untuk menandai lokasi wisata pada peta, Anda dapat menggunakan marker yang dapat dikustomisasi. Setiap marker dapat berisi:

  • Nama lokasi
  • Deskripsi singkat
  • Koordinat (longitude, latitude)
  • Popup informasi (dapat berisi gambar atau HTML lainnya)
new mapboxgl.Marker()
  .setLngLat([114.5915, -3.3172])
  .setPopup(
    new mapboxgl.Popup().setHTML(
      "<b>Taman Siring</b><br>Tempat wisata di Banjarmasin."
    )
  )
  .addTo(map);

Tips: Anda dapat menambahkan elemen HTML ke dalam popup seperti gambar, link, atau elemen lainnya sesuai kebutuhan.

Contoh Elemen HTML Peta

<div id="map" class="w-full h-96"></div>
