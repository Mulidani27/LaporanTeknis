Inisialisasi Peta Mapbox

Setelah Anda memperoleh token dari Mapbox, Anda dapat menginisialisasi peta menggunakan kode berikut:

  • ID elemen HTML tempat peta akan ditampilkan
  • Gaya peta (map style)
  • Koordinat pusat (longitude dan latitude)
  • Zoom awal saat peta dimuat
mapboxgl.accessToken = 'TOKEN_MAPBOX_ANDA';
const map = new mapboxgl.Map({
  container: 'map', // ID elemen HTML
  style: 'mapbox://styles/mapbox/streets-v11',
  center: [114.5944, -3.3186], // Koordinat Banjarmasin
  zoom: 12
});

Catatan: Gantilah TOKEN_MAPBOX_ANDA dengan token asli yang telah Anda dapatkan dari Mapbox.

Contoh Elemen HTML Peta

<div id="map" class="w-full h-96"></div>
